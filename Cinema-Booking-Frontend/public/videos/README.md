# Hero Background Video

Place your cinematic trailer video here as `hero-bg.mp4`.

## Requirements:
- Format: MP4 (H.264 codec recommended for best browser compatibility)
- Resolution: 1920x1080 or higher
- Duration: 15-30 seconds recommended (will loop)
- File size: Keep under 10MB for optimal loading

## Recommended Sources:
- Pexels Videos (free): https://www.pexels.com/videos/
- Pixabay Videos (free): https://pixabay.com/videos/
- Coverr (free): https://coverr.co/

## Search Terms:
- "cinema dark"
- "movie theater"
- "film reel"
- "cinematic abstract"
- "movie lights"

## Fallback:
If no video is placed, the hero section will display the featured movie's backdrop image instead.
